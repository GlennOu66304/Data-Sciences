## Python Flask

### Create the ven and indtall the Flask:
Create the ven
```
mkdir flask-tutorial
cd flask-tutorial
sudo apt-get install python3-venv
python3 -m venv venv
```
install the Flask
```
. venv/bin/activate
pip install Flask
```
